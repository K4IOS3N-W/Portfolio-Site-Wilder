# Resume Directory

This directory contains your resume file for download.

## Instructions:

1. Add your resume file here as: `Wilder_Machoco_Resume.pdf`
2. Make sure the file is in PDF format for best compatibility
3. The download link on your portfolio will point to this file

## Current Setup:

- The download button is located in the Curriculum section of your portfolio
- The link points to: `assets/resume/Wilder_Machoco_Resume.pdf`
- The button includes a download icon and hover effects

## To Update:

If you want to change the filename, update the `href` attribute in the download button in `index.html`:

```html
<a href="assets/resume/YOUR_NEW_FILENAME.pdf" class="btn btn-primary btn-lg" target="_blank" download>
```
